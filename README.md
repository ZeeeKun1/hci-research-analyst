# hci-research-analyst

An OpenClaw Skill for rigorous, theory-grounded data analysis in HCI research.

Designed for PhD students and researchers working toward top-tier HCI venues (CHI, CSCW, UIST, DIS, etc.).

## What It Does

- **7 Analysis Methods**: Thematic Analysis, Grounded Theory, Content Analysis, Discourse Analysis, Affinity Diagramming, Statistical Analysis, Mixed Methods
- **9 HCI Theories**: Cognitive Load, TAM/UTAUT, Presence & Immersion, Activity Theory, Affordance, Flow, Social Presence, Distributed Cognition, Constructivism
- **Paper-Ready Outputs**: Codebook template, Findings section template (CHI-style)
- **Bilingual Support**: English for paper content, Chinese for process explanations
- **Progressive Disclosure**: Lean SKILL.md core + detailed references loaded on demand

## Installation

### OpenClaw

Copy the entire hci-research-analyst/ folder to your OpenClaw skills directory:

`
<openclaw-skills-dir>/hci-research-analyst/
`

### Other Platforms

The SKILL.md and references/ structure is compatible with Claude Code, Cursor, and other AI coding assistants. Copy the folder to your agent's skills directory.

## File Structure

`
hci-research-analyst/
├── SKILL.md                          # Core workflow & method selection
├── references/
│   ├── methods/                      # Analysis method guides
│   ├── theory-library/               # HCI theory reference cards
│   └── templates/                    # Output templates
`

## Usage

Trigger the skill by asking your AI assistant to analyze research data, find themes, build a codebook, or prepare findings for a paper. See SKILL.md for the full workflow.
