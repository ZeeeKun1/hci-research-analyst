# HCI Theory Library — Index

This directory contains reference summaries of HCI-relevant theories. Each file provides: core constructs, key predictions, typical HCI applications, and foundational citations.

## When to Use

After identifying themes/patterns in Phase 4 of the analysis workflow, browse this index to find theories that illuminate your findings. Read specific theory files as needed.

## How to Match Theory to Findings

1. **Identify the core phenomenon** in your finding (e.g., "users felt overwhelmed by information" → cognitive load)
2. **Check the "Core Question" column** — does it align?
3. **Read the specific theory file** for detailed constructs and applications
4. **Map your finding to theory constructs** — which construct explains your observation?
5. **Check for tension** — does your finding contradict or extend the theory? That's often more valuable than simple confirmation.

## Extending This Library

To add a new theory:
1. Create a new `.md` file with the theory name
2. Follow the standard structure: Overview → Core Constructs → Key Predictions → HCI Applications → Foundational Citations
3. Add a row to this index table
4. Keep each file focused (one theory per file)
